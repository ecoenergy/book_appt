Elastix - Appointment Plus Integration (Book Appt. Button)

Add a click-able "Book Appt" link to the "Agent-Console" page in Elastix that works with Appointment-Plus in Chrome.

View on GitHub Download .zip Download .tar.gz

Description

A lightweight Chrome Extension that uses jquery to add a "Book Appt" button to the "agent-console" page in Elastix. This button updates each time a new call comes in.

Installation

To install, click on "download ZIP", then extract the contents. Open chrome, click on the "three lines" (top right), click "More Tools", and open "Extensions". Make sure developer mode is checked on, and then click "Load Unpacked Extension". Find the folder you just extracted, and you are off to the races.
